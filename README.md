# SimpleFlow - Sign Up & Sign In Page

SimpleFlow is a clean and simple sign-up/sign-in page designed for easy account registration and secure login. This page serves as the entry point for users to access the features of SimpleFlow.

# Features

 • Sign Up & Registration: Users can create an account by entering a username, email, and password.
 
 • Login Functionality: Returning users can securely sign in.
 
 • Responsive Design: The page is responsive and works seamlessly across all devices.
 
 # How to Use
 1. Sign Up / Sign In: Enter your username, full name, email, and password to create a new account.
 # How to View Locally
 To view the page locally:
 
 1. Clone this repository:https://github.com/parastoo-code/Simple-flow.git
 2. Open the index.html file in your web browser.
